Thumbor supports . Automagically.

WebP is a new format for lossy and lossless images that compresses a lot better than jpeg and png.

Thumbor allows users to specify that browsers that support WebP automatically get it. This can have a dramatic increase in the performance of your website.

Besides WebP, Thumbor also uses some tactics to reduce the file size even further. Use the input above to find how much bandwidth thumbor can save for you.

Light-headed? No more headless bodies.

Thumbor uses state-of-the-art face detection to ensure you get the best possible crop. Whenever a face is found it will crop your image around it making sure the most important parts of the photo are preserved.

Even though this process is expensive, Thumbor will serve the image with a "best possible" crop while the face detection is being performed. The next time the image is requested, the correct face-detected image is served instead.

Images here and there. Everywhere.

Thumbor allows users to store and load images from anywhere needed. It's really simple to implement a new loader or storage.

It comes packaged with file, http, redis and mongo implementations for loaders and storages. Just pick what makes sense for you or implement your own.

Thumbor has great documentation dedicated to help users get up to speed with it.
